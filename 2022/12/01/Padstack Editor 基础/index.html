基本

  1. Thru Pin 通孔
  2. SMD Pin 贴片
  3. Via 过孔
  4. BBVia 盲埋孔
  5. Microvia 微孔 （小于0.1mm的盲埋孔）
  6. Slot 槽孔

操作步骤

  1. start下选择 孔类型和焊盘风格
  2. Design Layers下 选择 SOLDERMASK_TOP 尺寸和 PASTEMASK_TOP尺寸
  3. 另存到自定义目录
  4. 保存名称 SOB0R45X0R45.pad (SMD OBLONG 0.45mm * 0.45mm )
