晶体管基本特性

1. 工作状态判断

  • 放大状态：

此时发射结正偏，集电结反偏。

  • 截至状态：

此时发射结0偏或反偏，集电结反偏。

  • 饱和状态：

主要特征： IBQ 增大时不会导致 ICQ 增大

一般情况认定发射结正偏，集电结反偏为饱和

  • 倒置状态：

此时发射结反偏，集电结正偏。（会使 β 下降严重）

环路稳定性判据

奈奎斯特稳定性判据，当系统的相位裕量大于0度时，此系统是稳定的。

  1. 在穿越频率处，总开环系统要有大于30度的相位裕量；
  2. 为防止-2增益斜率的电路相位快速变化，系统的开环增益曲线在穿越频率附近的增益斜率应为-1（ -20db/10倍频程)
  3. 增益裕量是开环系统的模的度量，该变化可能导致曲线刚好通过-1 点。一般需要6db的增益裕量。

备注：应当注意，并不是绝对要求开环增益曲线在穿越频率附近的增益斜率为必须为-1，但是由于-1增益斜率对应的相位曲线相位延迟较小，且变化相对缓慢，因此它能够保证，当某些环节的相位变化被忽略时，相位曲线仍将具有足够的相位裕量，使系统保持稳定。

开关电源基础

定义脉冲占空比：$ D = \frac{T_{on}}{T}$

T 表示开关S开关重复周期，$ T_{on} $ 表示开关S在一个开关周期中的导通时间。

直流输出电压 $U_o$ 输出电压与输入电压 $U_i$ 的关系：$U_o = U_i * D$

一、基本组成

  1. 驱动器
  2. 比较放大器
  3. 信号源
  4. 辅助电路 （ 过流过压保护，输入滤波，输出采样，功能指示）

二、开关电源特点

开关电源的特点 具体内容
效率高 功率开关管工作在开关状态，功耗小效率高。工作频率高在20KHz以上（滤波元件数值减低）
重量轻 节省了大量漆包线与硅钢片，使重量体积大幅下降
稳压范围宽 合理设计电源可以使输出电压范围在 $\pm2\%$ 以下
安全可靠 可以方便的设置各种形式的保护电路，负载出现故障时可以切断电源保证功能可靠

​ 开关电源的负载变化瞬态响应主要由输出端LC滤波器的特性决定，所以可以通过提高开关频率、降低输出滤波器LC的方法来改善瞬态响应特性。

​ 光电耦合器中三极管集电极电流 $I_C$ 的大小与光电二极管电流 $I_F$ 及光电耦合系数 $h$ 成正比： $I_C = h *I_F$

四、主要技术指标

技术指标 具体内容
输入电压变化范围 当输出电压保持固定时，输入电压的变化范围（表征适应外界变化的能力，电源使用范围）
【取决于电源的误差放大、反馈调节电路的增益、占空比调节的范围】
输出内阻 $ R_o$ 输出电压变化量 $\Delta U_o$ 与输出电流的变化量 $\Delta I_o$ 的比值。 $ R_o = \frac{\Delta U_o}{\Delta I_o} $
【$R_o$ 越小代表电源输出电压随负载电流的变化越小，稳压性能越好】
效率 $\eta$ 电源输出功率$P_o$与输入功率$P_i$的比值。$\eta = \frac{P_o}{P_i}$
【$\eta $ 越高，体积越小，可靠性越高】
输出纹波电压 表示方法：输出纹波有效值。输出纹波电压的峰峰值$U_{pp}$
【纹波电压越小，电源输出性能越好】
输出电压调节范围 输出电压取决于基准电压和取样电路的元件参数。
【反映在开关管调整脉冲占空比D的变化范围】
输出电压稳定性 输出电压随负载变化而变化的特性。（取决于反馈调节回路的增益及频率响应特性）
【反馈调节回路增益越高，基准电压$U_E$越稳定，输出电压$U_o$的稳定性就越好】
输出功率$P_o$ 电源能输出给负载的最大功率，与负载功率有关，要求存在 20-50%裕量

电源、地联通

1、系统自带电源/地连接符的库，不用添加，直接点击放置电源/地的按钮放置;
2、电源/地是全局变量，不同的页面也可以进行连接，不用添加分页符;
3、自定义电源/地连接符的Name名称，即可将原理图上电源/地全部联通。

添加BUS总线

1、总线代表的是具有相同电气特性的一-组导线，在具有相同电气特性的导线数目较多的情况下，可采用总线的方式;
2、点击键盘的B键放置总线，或者使用菜单栏中的Place- Bus功能,添加总线;
3、总线命名与网络标号的命名命令类似，通过点击N键或者或者点击菜单栏的Place- -Net Alias.. 命令可放置
网络标号。放置总线时的须注意:
●总线的名字不能是以数字结尾的;
●符号[ ]前后不能有空格;
●命名必须是名字加[ ]，如BUS[0:15]， BUS[0-15], BUS[0… 15]注意要有三点，三种中的一种;
4、信号线与总线的连接采用的是总线入口的方式，通过点击键盘的E键或者点击菜单栏中的Place- -Bug 命令。

2、在弹出的Create Differential Pair窗口中查找需要设置的网络名并添加到Selections中;

4、在②箭头处选择具体对应的网络名;

汇编指令

指令格式

==< opcode > { < cond > } { S } < Rd >, < Rn > { , < OP2 > }==

其中 < >中内容必不可少，{ }内容可省略

  1. < opcode >操作码，例如ADD表示算数加法
  2. { < cond> } 指令执行的条件域， 如EQ， NE
  3. { S }表示执行结果是否影响CPSR的值，使用后缀则影响
  4. < Rd > 表目的寄存器
  5. < Rn > 表示第一个操作数，为寄存器
  6. < OP2 > 表示第二个操作数，可以是立即数、寄存器或寄存器位移操作数
